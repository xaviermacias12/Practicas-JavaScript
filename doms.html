1. Manipulación del DOM con JavaScript

Ejercicio 1: Selección de Elementos del DOM

En este ejercicio, seleccionaremos elementos del DOM usando diferentes métodos.

Código JavaScript

// Seleccionar un elemento por ID
const myDiv = document.getElementById('myDiv');
console.log("myDiv:");
console.log(myDiv);

// Seleccionar elementos por clase
const elementsByClass = document.getElementsByClassName('myClass');
console.log("elementsByClass:");
console.log(elementsByClass);

// Seleccionar elementos por etiqueta
const elementsByTag = document.getElementsByTagName('div');
console.log("elementsByTag:");
console.log(elementsByTag);

// Seleccionar un elemento con querySelector
const queryElement = document.querySelector('#myDiv');
console.log("queryElement:");
console.log(queryElement);

// Seleccionar todos los elementos con querySelectorAll
const queryAllElements = document.querySelectorAll('div');
console.log("queryAllElements:");
console.log(queryAllElements);

Ejercicio 2: Manipulación de Contenido y Estilos

En este ejercicio, cambiaremos el texto y los estilos de un elemento del DOM.

Código JavaScript

const myDiv = document.getElementById('myDiv2');

// Cambiar el texto del elemento
document.getElementById('changeText').addEventListener('click', () => {
    myDiv.textContent = 'Text Changed!';
});

// Cambiar el estilo del elemento
document.getElementById('changeStyle').addEventListener('click', () => {
    myDiv.style.color = 'red';
    myDiv.style.fontSize = '24px';
    myDiv.style.fontWeight = 'bold';
});
Hello, World!

Resultado esperado:

Al hacer clic en "Change Text", el texto cambiará a "Text Changed!".

Al hacer clic en "Change Style", el texto cambiará a color rojo, tamaño de fuente 24px y negrita.

Ejercicio 3: Eventos del DOM

En este ejercicio, manejaremos eventos del DOM para interactuar con los elementos de la página.

Código JavaScript

const myDiv = document.getElementById('myDiv3');

// Manejar el evento click
myDiv.addEventListener('click', () => {
    alert('Div clicked!');
    myDiv.style.backgroundColor = 'lightblue';
});

// Manejar el evento mouseover
myDiv.addEventListener('mouseover', () => {
    myDiv.style.backgroundColor = 'yellow';
});

// Manejar el evento mouseout
myDiv.addEventListener('mouseout', () => {
    myDiv.style.backgroundColor = '';
});

Ejercicio 4: Crear una Página HTML y Manipular el DOM

En este ejercicio, crearemos una página HTML con varios elementos y utilizaremos JavaScript para realizar diversas manipulaciones.

Código JavaScript

// Cambiar el contenido del título
document.getElementById('changeContent').addEventListener('click', () => {
    document.getElementById('title').textContent = 'Content Changed!';
});

// Resaltar todos los párrafos
document.getElementById('highlightText').addEventListener('click', () => {
    const paragraphs = document.querySelectorAll('.text');
    paragraphs.forEach(p => {
        p.classList.toggle('highlight');
    });
});

Ejercicio 5: Ejercicio Completo de DOM

En este ejercicio, crearemos una página HTML completa y usaremos JavaScript para seleccionar elementos del DOM, manipular su contenido y estilos, y manejar eventos.

Código JavaScript

// Cambiar el contenido del encabezado
document.getElementById('changeHeader').addEventListener('click', () => {
    document.getElementById('mainTitle').textContent = '¡Título Cambiado!';
});

// Cambiar el contenido del párrafo
document.getElementById('changeDescription').addEventListener('click', () => {
    document.getElementById('description').textContent = 'La descripción ha sido modificada mediante JavaScript.';
});

// Cambiar los estilos de la lista
document.getElementById('styleList').addEventListener('click', () => {
    const listItems = document.querySelectorAll('.list-item');
    listItems.forEach(item => {
        item.style.color = 'green';
        item.style.fontWeight = 'bold';
    });
});

// Eventos mouseover and mouseout para elementos de lista
const listItems = document.querySelectorAll('.list-item');
listItems.forEach(item => {
    item.addEventListener('mouseover', () => {
        item.style.backgroundColor = 'lightgray';
    });
    
    item.addEventListener('mouseout', () => {
        item.style.backgroundColor = '';
    });
});
